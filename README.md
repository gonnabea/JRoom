Why?

CS이론을 공부하기 위한 효과적인 방법 중 하나는
기술을 먼저 직접 사용해서 무언가를 "어떻게든" 만들어 본 다음,
만들면서 생긴 의문이나 궁금한 점을 파고들어
역으로 개념을 추적하는 것이다.
위키백과는 이러한 학습을 하기에 적합한 구조를 가지고 있어
직접 클론해보며 여테 만들었던 프로젝트들과 관련된 이론을 스스로 정리해보고자 한다.
나 자신의 이론 공부가 주 목적이지만 실제로 개발을 하면서 만들기에
일석이조가 아닐까 생각이든다.
